# SoundBoard
SoundBoard simples utilizando JavaScript

<h3>Descrição do projeto</h3>
<p>Ao clicar nos botões, o tipo de som selecionado será emitido na página.</p>

<ul> 
  <li>HTML5</li>
  <li>CSS3</li>
  <li>BOOTSTRAP5</li>
  <li>JAVASCRIPT</li>
</ul>
